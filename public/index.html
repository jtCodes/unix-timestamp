API - Unix Timestamp Microservice

Features

Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)

If it does, it returns both the Unix timestamp and the natural language form of that date.

If it does not contain a date or Unix timestamp, it returns null for those properties.

Current Unix Time

Example Usage:

https://unix-date-timestamp.herokuapp.com/Aug%2005,%202017

https://unix-date-timestamp.herokuapp.com/1501905600

Example output:

{ "Unix": 1501905600, "Natural": "Aug 05, 2017" }
